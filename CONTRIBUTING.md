# Contributing

Thank you for your interest in contributing to Polysync.

## Reporting Issues

If you find a bug or encounter any unexpected behavior, please open an issue and include:

* Operating system
* Steps to reproduce
* Expected behavior
* Auctual behavior

## Pull Requests

Before opening a pull request:

* Ensure project compiles successfully
* Keep changes focues on a single feature or fix

## Questions

If you are unsure about a design decision or implementation approach, feel free to open an issue or discussion before beginning work.
